BUILD THE KALI IMAGE ON A MICRO SD CARD:
1. Download kali linux: https://www.offensive-security.com/kali-linux-vmware-arm-image-download/
2. Verify the download: shasum kali<insert full filename here>
3. Insert micro SD card into computer for imaging and determine the appropriate device path:
> diskutil list
4. Unmount the appropriate disk:
> diskutil unmount /dev/disk1
5. Use dd to image the kali file to the card:
dd if=kali-linux-1.1.0-rpi2.img of=/dev/disk1 bs=1m
(control+T to see progress of image creation)
6. Eject the micro SD card
7. Insert disk into Raspberry Pi 2, plug in peripherals and power it up. Voila!

References:
http://docs.kali.org/kali-on-arm/install-kali-linux-arm-raspberry-pi
http://docs.kali.org/downloading/kali-linux-live-usb-install

CHANGE SSH HOST KEYS:
root@kali:~ rm /etc/ssh/ssh_host_*
root@kali:~ dpkg-reconfigure openssh-server
root@kali:~ service ssh restart

TIME SETUP:
dpkg-reconfigure tzdata

SET UP VNC ACCESS:
Reference: http://hezik.nl/remote-control-your-raspberry-pi-running-kali-linux-using-vnc-server/
apt-get update
apt-get dist-upgrade -y
apt-get install x11vnc -y
startx &
x11vnc & (in a separate terminal window)
Then, launch a VNC app (Chicken of the VNC or Remmina?) to connect.

SET UP REMOTE DESKTOP:
Reference: http://www.linux.org/threads/rdp-into-your-linux-desktop.8/
yum install xrdp -y
chkconfig --levels 35 xrdp on
service xrdp start
service xrdp stop (when finished)

SET UP WIFI:
Reference: http://www.freehowtohackwifi.com/general-wifi-hacking/enable-wifi-kali-linux/
iwconfig
kfconfig wlan0 up
Use the GUI: Now that the wireless adapter is up, click the icon of the two computers at the upper right-hand corner of the screen. The drop down box will display all the wireless networks that are within range, and you can connect to your target after successfully applying the wireless password.
